# AI 直播复盘分析 — 部署说明（开箱即用便携版）

这是打包好的便携版。目标电脑**不需要预装 Python、不需要管理员权限**，
所有运行环境（Python + 依赖库 + ffmpeg）都装在本文件夹内，删掉文件夹即彻底卸载。

---

## 在目标电脑上怎么用（两步）

1. **首次安装**：双击 `install.bat`
   - 会自动下载便携版 Python、程序依赖、ffmpeg（约 150MB，需联网，只需一次）。
   - 看到「全部安装完成！」即可。
   - 中途如果报错，多半是网络访问不了 GitHub，挂上代理重试。

2. **运行**：双击 `启动.bat`
   - 自动打开浏览器 `http://localhost:8501`。
   - 关闭弹出的黑色命令行窗口 = 关闭程序。

API 密钥（众盟 / Anthropic / Google）已经随包打好，开箱即用。
密钥失效时，在程序首页「⚙️ API 配置」面板里更新即可。

---

## ⚠️ 唯一一个无法打包的依赖：本地代理

程序的**下载录像**和**Gemini 看录像**需要走海外代理（默认 `http://127.0.0.1:7890`，
即 Clash / V2Ray 这类工具的本地端口）。这个东西装在你电脑的网络层，没法塞进包里。

- 如果目标电脑**已经有代理**且端口也是 `7890`：什么都不用做。
- 如果端口不同：在程序首页「⚙️ API 配置」里改 `LOCAL_PROXY`（如 `http://127.0.0.1:你的端口`）后保存。
- 如果目标电脑**没有代理**：数据抓取、复盘文字分析（走的是国内可达的代理域名）一般能用；
  但**下载录像 / 录像视觉分析**会失败。需要先在那台电脑装好代理工具。

> 「声音」维度的音频转写（faster-whisper）默认未打包以缩小体积。
> 需要时在本文件夹运行：`runtime\python\python.exe -m pip install faster-whisper`

## 目录结构

```
ai_analysis_portable/
├─ install.bat          ← 第一步：双击安装环境
├─ 启动.bat             ← 第二步：双击运行
├─ requirements.txt     依赖清单
├─ 部署说明.md          本文件
├─ packaging/setup.ps1  安装逻辑（被 install.bat 调用）
├─ app/                 程序本体（含 config.json 密钥）
│  ├─ app.py, pages/, *.py
│  ├─ config.json       你的 API 密钥
│  └─ data/             数据/缓存（运行后自动生成）
├─ runtime/python/      便携版 Python（install 后生成）
└─ ffmpeg/bin/          ffmpeg.exe（install 后生成）
```

## 给开发者：如何重新生成这个包

在源机器（本机 `ai_analysis` 目录）运行：

```powershell
powershell -ExecutionPolicy Bypass -File packaging\make_package.ps1
```

产物在 `_dist\ai_analysis_portable.zip`，把它发给对方即可。

## 常见问题

- **install.bat 一闪而过 / 下载失败**：通常是连不上 GitHub（Python 包托管在那）。挂代理后重试。
- **启动.bat 提示「未安装」**：先跑 install.bat。
- **录像分析报「抽帧失败」**：ffmpeg 没装好，重跑 install.bat。
- **录像下载失败**：检查本地代理是否开启、端口是否对（见上方代理说明）。
- **端口 8501 被占用**：编辑 `启动.bat`，把 `--server.port=8501` 改成别的端口。
